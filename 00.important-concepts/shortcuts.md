# ⌨️  Essential keyboard shortcuts

Learning keyboard shortcuts is one of the fastest ways to improve your efficiency and confidence when using a computer. Instead of relying on slow, repetitive mouse actions, shortcuts help you complete tasks quickly, stay focused, and work more professionally. As a habit, whenever you start using or download any new software, **make it a priority to learn its essential shortcuts first**. This small effort will save you a significant amount of time in the long run and make your overall workflow smoother and more productive.

## 🧠 Basic Editing

- **Copy** → `Ctrl + C` (imp)
- **Paste** → `Ctrl + V` (imp)
- **Cut** → `Ctrl + X`
- **Undo** → `Ctrl + Z`
- **Redo** → `Ctrl + Y`
- **Select all** → `Ctrl + A` (imp)
- **Find** → `Ctrl + F`

## 📄 File & Document

- **Save** → `Ctrl + S` (imp)
- **Open file** → `Ctrl + O`
- **New file/window** → `Ctrl + N`
- **Print** → `Ctrl + P`

## 🖥️ System Shortcuts

- **Switch apps** → `Alt + Tab` (imp)
- **Close window** `Alt + F4` (imp)
- **Lock computer** → `Win + L`
- **Screenshot (full)** → `PrtScn`

## 🌐 Google Chrome Shortcuts

### 🔖 Tabs & Windows

- **New tab** → `Ctrl + T` (imp)
- **Close tab** → `Ctrl + W` (imp)
- **Next / Previous tab** → `Ctrl + PageUp` / `Ctrl + PageDown`
- **Jump to tab (1–8)** → `Ctrl + 1–8`
- **Last tab** → `Ctrl + 9`

### 🔍 Navigation

- **Go to address bar** → `Ctrl + L` (imp)
- **Open new window** → `Ctrl + N` (imp)
- **Incognito mode** → `Ctrl + Shift + N`
- **Back / Forward** → `Alt + Left / Right`
- **Reload page** → `Ctrl + R` or `F5`
- **Empty cache / Hard reload** → `Ctrl + Shift + R` (imp)

### ⚡ Productivity

- **Downloads** → `Ctrl + J`
- **History** → `Ctrl + H`
- **Bookmark page** → `Ctrl + D`
- **Open bookmarks bar** → `Ctrl + Shift + B`

## 💻 VS Code Shortcuts

### 📂 File & Editor

- **Close editor** → `Ctrl + W`

### 🔎 Navigation & Search

- **Quick open file** → `Ctrl + P`
- **Search in project** → `Ctrl + Shift + F`
- **Go to line** → `Ctrl + G`
- **Go to symbol** → `Ctrl + Shift + O`

### ✍️ Editing Power

- **Move line up/down** → `Alt + ↑ / ↓` (imp)
- **Multi-cursor** → `Alt + Click`
- **Select next match** → `Ctrl + D` (imp)
- **Duplicate line** → `Shift + Alt + ↓` (imp)
- **Select all matches** → `Ctrl + Shift + L`
- **Command palette** → `Ctrl + Shift + P` ⭐ (very powerful)

### 🧠 Code Intelligence

- **Comments** → `Ctrl + /`
- **Format document** → `Shift + Alt + F` (very imp)
- **Auto-complete trigger** → `Ctrl + Space` (imp)
- **Quick fix / suggestions** → `Ctrl + .`
- **Rename variable** → `F2`

### 🖥️ Terminal & Panels

- **Toggle terminal** → `` Ctrl + ` ``
- **New terminal** → ``Ctrl + Shift + ` ``
- **Toggle sidebar** → `Ctrl + B`

## ⚡ Pro Tips

- To Learn First (High Impact) memorize a few :
  - Chrome: `Ctrl + T`, `Ctrl + L`
  - VS Code: `Ctrl + P`, `Alt + ↑/↓`, `Ctrl + D`

- Learn **Ctrl combos first** — they’re used everywhere
- Practice a few daily instead of memorizing all at once
- Most apps follow the same patterns
