# Web Development Course

## Class code repo for students `SMIT` Batch 22

Gulshan campus 5pm to 7pm TTS | Mar 2026 - 2027

## 🎯 Course Objectives

_By the end of this course, students will be able to:_

- Be a `problem solver`.
- Learn how to `think`.
- Improve communication and confidence.
- Build online presence.

- Programming
  - **Understand core concepts**
    Grasp fundamental concepts of programming and logic.
  - **Write clean and working code**
    Develop the ability to write readable, structured, and error-free code.
  - **Build real-world projects**
    Apply concepts to create practical projects (e.g., websites or apps).
  - **Debug and solve problems**
    Identify errors and fix them using logical thinking and debugging techniques.
  - **Think like a developer**
    Break problems into smaller steps and approach solutions systematically.
  - **Collaborate and learn independently**
    Work with others, ask good questions, and continue learning beyond the course.
